# Project 2 – AWS EC2 Web Server

## 📌 Objective
Launch an Amazon EC2 instance, install and configure the **Nginx Web Server**, and access the default webpage using the EC2 Public IP.

## 🛠️ AWS Services Used
- Amazon EC2
- Nginx Web Server

## ⚙️ Instance Details

| Property | Value |
|----------|-------|
| Instance Type | t3.micro |
| Operating System | Amazon Linux |
| Region | us-east-1 |

> **Note:** If you used `t2.micro`, replace the instance type accordingly.

## 📋 Steps Performed

1. Launched an Amazon EC2 instance.
2. Connected to the instance using **SSH**.
3. Updated the system packages.
4. Installed the **Nginx Web Server**.
5. Started and enabled the Nginx service.
6. Configured the Security Group to allow **HTTP (Port 80)** traffic.
7. Accessed the default Nginx webpage using the **EC2 Public IPv4 Address**.

## 💻 Technologies Used

- Amazon EC2
- Amazon Linux
- Nginx
- SSH
- Linux Command Line
  
## 🚀 Features

- Virtual cloud server deployment
- Nginx web server installation
- Remote server management using SSH
- Public web access through EC2 Public IP
- Security Group configuration for HTTP traffic.

## ✅ Result

Successfully launched an **Amazon EC2** instance, installed and configured the **Nginx Web Server**, and verified the deployment by accessing the default Nginx webpage through the **EC2 Public IPv4 Address**.
