# Project 3 – AWS RDS MySQL Database

## 📌 Objective

Create and configure an **Amazon RDS MySQL** database instance and make it available for database applications.

## 🛠️ AWS Services Used

- Amazon RDS
- MySQL

## 🗄️ Database Details

| Property | Value |
|----------|-------|
| Database Name | project3-db |
| Engine | MySQL Community |
| Instance Type | db.t3.micro |
| Region | us-east-1 |
| Port | 3306 |
| Status | Available |

## 📋 Steps Performed

1. Created an Amazon RDS MySQL database instance.
2. Configured database settings and allocated storage.
3. Selected the **db.t3.micro** instance type.
4. Configured networking, VPC, and Security Group settings.
5. Waited for the database instance status to become **Available**.
6. Verified that the database instance was running successfully.

## 💻 Technologies Used

- Amazon RDS
- MySQL
- AWS Management Console
  
## 🚀 Features

- Fully managed relational database service
- MySQL Community Edition
- Secure networking with VPC and Security Groups
- Automated database deployment
- Scalable and reliable cloud database

## ✅ Result

Successfully created and deployed an **Amazon RDS MySQL** database instance on AWS. The database was configured with the required networking and security settings, reached the **Available** state, and is ready to be used by applications.
