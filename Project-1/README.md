# Project 1 – AWS S3 Static Website Hosting

## 📌 Objective
Host a static portfolio website using **Amazon S3** and make it publicly accessible over the internet.

## 🛠️ AWS Services Used
- Amazon S3

## 📋 Steps Performed
1. Created an Amazon S3 bucket.
2. Uploaded the website files (HTML, CSS, and other assets).
3. Enabled **Static Website Hosting** for the bucket.
4. Configured bucket permissions to allow public access.
5. Generated the **S3 Website Endpoint**.
6. Verified that the website was successfully accessible through the S3 Website URL.

## 💻 Technologies Used
- HTML
- CSS
- Amazon S3

## 🚀 Features
- Static website hosting
- Public website accessibility
- Fast and reliable content delivery using Amazon S3
- Simple and cost-effective hosting solution
